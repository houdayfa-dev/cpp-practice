# Stone-Paper-Scissors Game
**Author:** Houdayfa Ezwini

## Description
This is a C++ console program to play Stone-Paper-Scissors against the computer.  
It allows the player to choose how many rounds to play and reports the winner of each round and the overall game.

## How it Works
- The player selects a number corresponding to Stone (1), Paper (2), or Scissors (3).  
- The computer randomly selects its choice.  
- The program compares both choices and declares the winner of the round.  
- At the end, a final report shows:
  - Player wins
  - Computer wins
  - Draws
  - Final winner

## How to Run
1. Compile the code in the terminal:
   `g++ src/Project1_Stone_Paper_Scissors.cpp -o build/game.exe`
2. Run the game:
   `./build/game.exe`
